nanoDDS

nanoDDS is an Open Source Data Distribution Service (DDS) for embedded devices implemented from scratch to provide an open alternative in the embedded world for ROS 2, the next generation of the Robot Operative System framework. Refer to this repository for the code.

Chapter 1: Background provides a background research and comparison betweem different embedded DDSs. Chapter 2: DDS will describe and document the implementation of nanoDDS. Chapter 3: Using nanoDDS will show how to use nanoDDS through a set of tutorials and examples.
